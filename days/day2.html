It's "Advent Code Time" for whatever reason we went from 50 challengers to one! But conveniently that is all you need to decorate the Christmas tree. That star you can collect by completing the following puzzle!

Due to the Industrial Revolution, the tree's additional decorations are automated. The issue is that the engineer Alston Liu, TIME TRAVELED TO LAS VEGAS! So there are some errors which make decorations of the same color appear next to each other when they should be spread out!

We will use the following input as an example.

123456

0123456

654321

123

1. You have a list of numbers in each row

2. You will have to take the first number of the first line
ex. 1

3. You will have to take the second number of the second line
ex. 1

4. Then compare those numbers if they are equal
ex. 1 They are equal!

5. Repeat steps 2 and 3 with the next lines such as...

* You take the third number of the third line

* Then compare this new number with the previous most recent number :D

* If the index of the number you take is not available, wrap around the same line such as...

- if the current line is "123456" but we want to take the 7th number, we would take the first number which is "1"

6. The final answer will be the number of times the two numbers are equal!

* In the example it would be one!
